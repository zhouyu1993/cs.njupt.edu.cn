下面针对性进行说明。

# 关于后台上传的图片，建议需要显示图片的栏目，其文章中的第一张图片，一律采用 .jpg 格式。
原因：
* .jpg 格式支持性大，压缩比小
* 有些模块，webplus 会转化图片格式，例如 .png 会转化成 .jpg 。这样导致图片无法显示。
* 关于图片大小问题，目前后台手动设定的显示原图，所以在上传图片的时候，请规范图片尺寸，如出现问题，请即时反馈。
* 一般来说，大图需要进行压缩。推荐网站 https://tinypng.com/ 。或者 PS 自行压缩。

# 关于文章主体内容格式
源码中未写太多文本修饰，只是设定了默认字体大小，16px。具体格式，后台编辑时请自行定义
